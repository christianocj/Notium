# Gestão de conhecimento pessoal (KnowledgeHub)

## Conceito Geral
Uma API de gestão de conhecimento pessoal onde o usuário pode armazenar anotações, resumos de estudos, documentações, links importantes e aprendizados. O objetivo é centralizar informações e facilitar sua consulta futura através de organização e pesquisa.

É um projeto simples de entender, útil para o dia a dia e excelente para praticar desenvolvimento de APIs profissionais com .NET. Na prática, o usuário entra no sistema, cria uma nota, classifica essa nota dentro de uma categoria, adiciona algumas palavras-chave para facilitar futuras buscas e depois consegue encontrá-la rapidamente quando precisar

## Objetivo
O projeto foi criado para praticar conceitos modernos de desenvolvimento backend utilizando o ecossistema .NET, simulando um cenário real de aplicação com autenticação, persistência de dados, arquitetura em camadas e deploy em nuvem.

## Funcionalidades Essenciais
   * Cadastro de usuários
   * Autenticação JWT
   * Gerenciamento de categorias
   * Gerenciamento de notas
   * Pesquisa de conteúdo
   * Arquivamento de notas
   * Paginação e filtros

## Arquitetura
O projeto segue os princípios de Clean Architecture, separando responsabilidades entre:
   * API
   * Application
   * Domain
   * Infrastructure

## Tecnologias
   * .NET 10
   * ASP.NET Core
   * SQL Server
   * Entity Framework Core
   * JWT Authentication
   * Scalar/OpenAPI
